# Frontend Documentation

Este proyecto utiliza React para el frontend y está diseñado para interactuar con un backend basado en Node.js y SQL.

## Estructura del Proyecto

- **public/index.html**: Plantilla HTML principal que carga la aplicación React.
- **src/App.js**: Componente principal que configura la estructura de la aplicación.
- **src/components/**: Contiene componentes reutilizables de la aplicación.
- **src/pages/**: Contiene las diferentes páginas de la aplicación.
- **src/styles/App.css**: Estilos CSS utilizados en la aplicación.

## Instalación

1. Asegúrate de tener [Node.js](https://nodejs.org/) instalado en tu máquina.
2. Navega al directorio del frontend:
   ```
   cd frontend
   ```
3. Instala las dependencias:
   ```
   npm install
   ```

## Ejecución

Para iniciar la aplicación en modo de desarrollo, ejecuta:
```
npm start
```

Esto abrirá la aplicación en tu navegador por defecto.
